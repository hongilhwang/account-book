# Account book ( 가계부 )
가계부를 작성 하는 프로그램을 만드는 프로젝트 입니다.
개인 프로젝트이며, Electron에 대해서 공부하고, React를 학습하기 위한 의도입니다. 
다른 거창한 뜻은 없으므로, 프로그램이 정상 작동 안하고, 프로젝트가 진행이 안되는 경우가 대부분입니다. 
개인의 소소한 놀이를 위한 프로젝트임을 밟힙니다.

CRA + Eelectron + rescripts

## 배포 
```$xslt
npm run  electron-pack
```

## 개발 실행 
```$xslt
npm run  electron-dev
```
